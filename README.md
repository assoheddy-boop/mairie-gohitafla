# Mairie de Sikensi - Site Officiel

Site internet officiel de la Mairie de Sikensi, Côte d'Ivoire.

## Fonctionnalités

- Page d'accueil avec présentation de la commune
- Présentation de la mairie et du conseil municipal
- Services en ligne : demande de documents administratifs
- Guide des démarches administratives
- Actualités et événements de la commune
- Formulaire de contact

## Documents disponibles en ligne

- Acte de naissance
- Acte de mariage
- Acte de décès
- Certificat de résidence
- Légalisation de documents
- Certificat de célibat

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React (icônes)

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
